AskTheMap is an AI agent for local search and planning. The system allows users to interact through natural language to search for places, plan routes, discover local events, and organize activities.

The project integrates large language models with external tools such as maps, search APIs, and calendar-related workflows. It reflects my interest in building practical AI agents that can understand user intent, coordinate tools, and support real-world decision-making.

**Keywords:** AI agents, tool use, local search, maps, planning, human-AI interaction
